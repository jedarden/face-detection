How it works

This demo uses face-api.js, a JavaScript face detection library built on TensorFlow.js. It runs entirely in your browser using WebGL acceleration - no data is sent to any server.

Features:

  • Real-time face detection with bounding boxes
  • Optional facial landmark detection (68 points)
  • Expression recognition (happy, sad, angry, etc.)
  • Age and gender estimation
  • Multiple camera support
  • Two detection models for speed vs accuracy trade-off

Privacy:

All processing happens locally in your browser. No images or data leave your device.
